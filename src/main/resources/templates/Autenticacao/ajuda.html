Básico Tarefas Focus Mode

Funcionamento Básico

Adicionar Tarefas

Para adicionar uma nova tarefa:

  1. Digite o nome da tarefa no campo principal
  2. Clique no botão "Adicionar" ou pressione Enter
  3. Para opções avançadas, clique em "Mais opções"

Dica: Use atalhos! Pressione Enter para adicionar rapidamente uma tarefa simples.

Gerenciamento de Tarefas

Completar Tarefas

Para marcar uma tarefa como concluída:

  • Clique no ícone de check à esquerda da tarefa
  • A tarefa será riscada e marcada como concluída
  • Clique novamente para desmarcar

Editar Tarefas

Para editar uma tarefa existente:

  • Clique no ícone de lápis (editar)
  • O formulário será preenchido com os dados atuais
  • Modifique o que precisar e clique em "Salvar"

Excluir Tarefas

Para remover uma tarefa:

  • Clique no ícone de lixeira
  • Confirme a exclusão na janela de diálogo
  • A ação não pode ser desfeita

Focus Mode (Pomodoro)

Como usar

O Focus Mode ajuda você a se concentrar em uma única tarefa usando a técnica Pomodoro:

  1. Selecione uma tarefa clicando nela (ela ficará destacada)
  2. Clique no botão "Focus Mode" no canto superior esquerdo
  3. Use os controles para gerenciar seu tempo:
    • Iniciar: Começa a contagem regressiva de 25 minutos
    • Pausar: Pausa o temporizador
    • Parar: Reinicia o temporizador
  4. Quando o tempo acabar, você será notificado

Notificações

O sistema solicitará permissão para mostrar notificações quando o tempo do Pomodoro acabar.

Certifique-se de permitir notificações no seu navegador para receber os avisos.

Sons

Um som de alarme será reproduzido quando o tempo acabar.

Verifique se o volume do seu dispositivo está ajustado adequadamente.

Dúvidas Frequentes

Sim, todas as suas tarefas são armazenadas localmente no seu navegador usando o localStorage. Isso significa que:

  • Seus dados não são enviados para nenhum servidor
  • As tarefas persistem mesmo se você fechar o navegador
  • Os dados ficam disponíveis apenas no dispositivo onde foram criados

Atualmente não, pois o sistema foi projetado para armazenar dados apenas localmente. Cada dispositivo terá sua própria lista independente.

Você pode exportar suas tarefas seguindo estes passos:

  1. Abra as Ferramentas de Desenvolvedor (F12)
  2. Vá para a aba "Application"
  3. Selecione "Local Storage" no menu lateral
  4. Copie o conteúdo da chave "tasks"

Estamos trabalhando em uma solução mais amigável para exportação/importação.
